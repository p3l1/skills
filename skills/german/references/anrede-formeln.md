# Anrede und Grußformeln

## Anrede-Regeln

### Formell (Standard für Erstkontakt)
- **Unbekannt:** "Sehr geehrte Damen und Herren,"
- **Name bekannt:** "Sehr geehrte Frau Müller," / "Sehr geehrter Herr Schmidt,"
- **Akademischer Titel:** "Sehr geehrte Frau Dr. Müller,"

### Halb-formell (Nach erstem Kontakt)
- "Liebe Frau Müller," / "Lieber Herr Schmidt,"
- "Guten Tag Frau Müller,"

### Informell (Bei etabliertem Kontakt)
- "Hallo Thomas,"
- "Hi Sarah,"

## Grußformeln

### Formell
- "Mit freundlichen Grüßen"
- "Mit besten Grüßen"

### Halb-formell
- "Beste Grüße"
- "Herzliche Grüße"
- "Viele Grüße"

### Informell
- "Grüße"
- "VG" (nur bei sehr engem Kontakt)

## Wann welche Stufe?

| Situation | Anrede | Grußformel |
|-----------|--------|------------|
| Erstkontakt, unbekannt | Sehr geehrte/r | Mit freundlichen Grüßen |
| Erstkontakt, Name bekannt | Sehr geehrte/r + Name | Mit besten Grüßen |
| Nach 1-2 Kontakten | Liebe/r + Name | Beste Grüße |
| Regelmäßiger Austausch | Hallo + Vorname | Beste Grüße |
| Enger Kontakt | Hi + Vorname | Grüße |
