# skills

Personal Claude Code plugin marketplace.

## Installation

Add to `~/.claude/settings.json`:

```json
{
  "enabledPlugins": {
    "skills@skills": true
  },
  "extraKnownMarketplaces": {
    "skills": {
      "source": {
        "source": "github",
        "repo": "p3l1/skills"
      }
    }
  }
}
```
